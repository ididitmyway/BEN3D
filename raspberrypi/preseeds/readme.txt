Simple How to Raspberry Pi
- download offical raspian image
- download Win32Diskimager (for Windows)
- burn image on sd
- add ssh file for ssh access (headless config)
- add wpa_supplicant.con in case to preconfigure wlan access
- add sd to pi and boot

"ssh"
-> add a empty "ssh" file to the raspberry pi root directory to add ssh on first start of the pi

"wpa_supplicant.con"
-> add this to raspberry pi root  (replace the cornerning values with your wlan setting)
